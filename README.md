# Sheng Cao Personal Website

This repository contains the source code for my personal website.

Website: [shengcaotec.top](https://shengcaotec.top)

## Structure

- `index.html` - Home page
- `CV.html` - Curriculum vitae page
- `blog.html` - Blog index page
- `pictures.html` - Photo gallery page
- `assets/css/` - Page styles and shared design tokens
- `assets/js/` - Shared scripts and blog loading logic
- `assets/data/blog/` - Blog article HTML fragments
- `assets/images/` - Icons and general images
- `assets/photos/` - Project and life photos
